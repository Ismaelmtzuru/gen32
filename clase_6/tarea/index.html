<head>

Define la sección de encabezado de la página, donde se incluyen elementos como el título de la página, metaetiquetas, scripts, entre otros.

<meta charset="UTF-8">
<title>Mínimo Ejemplo de Etiqueta head</title>

<title>

Define el título de la página web que aparece en la pestaña del navegador.

<title>texto que aparece en pestaña del navegador</title>

<body>

Define la sección del cuerpo de la página web, donde se incluyen todos los elementos que se mostrarán en la página.

<body> aqui va el contenido del body </body>

<h1>-<h6>

Define los encabezados o títulos de diferentes niveles de jerarquía en la página web. El uso de esta etiqueta formateará cualquier texto entre la etiqueta <h> de apertura y la etiqueta </h> de cierre como un Título o subtítulo. Por ejemplo, en <h>Etiqueta de encabezamiento</h2>, <h2> y </h2> serían las etiquetas HTML y "Etiqueta de encabezamiento" es el elemento HTML, es decir, el encabezamiento de la página.

<a>

Define un enlace que el usuario puede hacer clic para ir a otra página web o a una sección diferente de la misma página.

<img>

Define una imagen que se mostrará en la página web.

<img> src="https://picsum.photos/100/100"

<p>

Define un párrafo de texto. Las etiquetas <p> y </p> son etiquetas HTML y la "Etiqueta de párrafo" es el elemento HTML, es decir, el texto de la página. Esta etiqueta formatea cualquier texto entre la etiqueta <p> de apertura y la etiqueta </p> de cierre como un párrafo estándar o texto de cuerpo principal.

<p> Aquí va el texto del parrafo </p>

<b>

Esta etiqueta dará formato de negrita a cualquier texto que se encuentre entre la etiqueta <b> de apertura y la etiqueta </b> de cierre. Aquí, <b> y </b> son las etiquetas HTML y la "etiqueta de negrita" es el elemento HTML, es decir, el texto de la página.

Se escribe: <b> texto en negrita </b>
Se visualiza: texto en negrita

<i>

Esta etiqueta formateará como cursiva cualquier texto entre la etiqueta de apertura <i> y la etiqueta de cierre </i>. Aquí, <i> y </i> son las etiquetas HTML y "Etiqueta cursiva" es el elemento HTML (el texto de la página).

Se escribe: <i> texto en cursiva </i>
Se visualiza: texto en cursiva

<u>

Esta etiqueta formateará cualquier texto entre la etiqueta <u> de apertura y la etiqueta </u> de cierre como subrayado. Aquí, <u> y </u> son las etiquetas HTML y "Etiqueta de subrayado" es el elemento HTML, es decir, el texto de la página.

Se escribe: <u> texto subrayado </u>
Se visualiza: texto subrayado

<strong>

Indica que el texto dentro de él, es especialmente importante o relevante, y se formatea generalmente en negrita.

Se escribe: Esto es un texto <strong> importante </strong>
Se visualiza: Esto es un texto importante

<em>

Indica que el texto dentro de él, deebe ser enfatizado, y se formatea generalmente en cursiva

Se escribe: Este es un ejemplo de <em> enfatizador </em>
Se visualiza: Este es un ejemplo de enfatizador

<abbr>

Representa una abreviatura o acrónimo, y el atributo 'title' proporciona una descripción completa

Se escribe: Este es un ejemplo de abreviación <abbr title="Hypertext Markup Language"> HTML </abbr>
Se visualiza: Este es un ejemplo de abreviación HTML
